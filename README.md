# Practica_Debian

Comandos Debian (Linux)
pwd: Muestra carpeta actual

whoami: Muestra usuario logueado

hotsname: Muestra nombre del servidor actual

date: Me da fecha y hora actual

ls: Muestra archivos actuales

ls -l: Muestra archivos de otra forma

ls -la: Muestra archivos ocultos

ls -lah: Muestra archivos ocultos con mejor presentación

mkdir: hace carpeta

cd NOMBRE/: Me mueve a la carpeta nombre

cd .:Me ubica en la carpeta actual, sirve para ir a la carpeta desde un archivo dentro de la misma carpeta

cd ..: Me devuelve una carpeta

cat ./NOMBRE_ARCHIVO:Abre un archivo y me muestra lo que tiene dentro

cat../NOMBRE_ARCHIVO:Abre un archivo y me muestra lo que tiene en la carpeta anterior

mkdir sub-folder: Crea una carpeta sub-folder

mv NOMBRE-ARCHIVO/ NOMBRE_NUEVO: Con mv renombro archivos

clear: Limpia la terminal

touch notas.txt: Crea archivo notas.txt

echo "HOLA MUNDO": Imprime Hola mundo en la terminal

echo "HOLA MUNDO" > notas.txt: Imprime hola mundo en archivo notas.txt, reemplazando el contenido de adentro(>)

echo "HOLA MUNDO" >> notas.txt: Imprime dentro de notas.txt hola mundo, pero no reemplaza sino que hace una nueva línea.

cp notas.txt notascopia.txt:Copia la información de notas.txt a notascopia.txt

cp notas.txt sobfolder/notas4.txt: Copia notas dentro del subfolder en el archivo notas4

rm notas.txt:Elimina notas.txt

top: muestra los procesos que está corriendo en el servidor

tf -h: muestra el disco del equipo

free -h:Muestra memoria total, libre y ocupada en el servidor

uname -a: muestra sistema operativo que está corriendo

history: Muestra historial de comandos

rm -r Folder 1/:Elimina la carpeta Folder 1 con los archivos que tenga dentro

ip a: Muestra interfaces de internet que tiene la maquina puesta.
