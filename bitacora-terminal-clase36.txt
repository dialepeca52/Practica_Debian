8. Comandos ejecutados y resultado observado:

Comando: whoami
Resultado: Muestra ususario actual

Comando: hostname
Resultado: Muestra nombre server

Comando: date
Resultado: muestra fecha y hora actual

Comando: pwd
Resultado: muestra carpeta donde estoy

Comando: ls
Resultado: muestra archivos de esa carpeta

Comando: ls -la
Resultado: Muestra archivos ocultos

Comando: mkdir practica-linux
Resultado: crea carpeta practica-linux

Comando: cd practica-linux
Resultado:cambia a carpeta practica-linux

Comando: touch notas.txt
Resultado: crea archivo notas.txt

Comando: echo "Mi primera practica en terminal Debian" > notas.txt
Resultado: Imprime ese texto en archivo notas .txt

Comando: cat notas.txt
Resultado: muestra lo que hay en el archivo notas.txt

Comando: cp notas.txt copia-notas.txt
Resultado: copia notas.txt a copia-notas.txt

Comando: mv copia-notas.txt notas-renombradas.txt
Resultado:Renombra archivos

Comando: mkdir respaldo
Resultado: crea carpeta respaldo

Comando: mv notas-renombradas.txt respaldo/
Resultado: cambia el nombre a archivo notas-renombradas.txt en carpeta respaldo

Comando: df -h
Resultado: Muetsra el disco del equipo

Comando: free -h
Resultado: Muestra memoria

Comando: uname -a
Resultado: Muestra sistema operativo corriendo

Comando: history
Resultado: Muestra historial de comandos

Comando: ip a
Resultado: Muestra interfaces de internet que tiene la maquina puesta

Comando: top
Resultado: Muestra procesos corriendo en el servidor
